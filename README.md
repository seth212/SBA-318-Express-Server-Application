PROJECT NAME: AutoExplorer Express Server Application SBA

DESCRIPTION:
AutoExplorer is an Express.js application for managing a small car marketplace. 

The project includes RESTful API routes, EJS views, custom middleware, error handling, static CSS, and form submission.

FEATURES:

- View all cars
- View a single car by ID
- Filter cars by make using query parameters
- Add new cars through a form
- Update car information
- Delete cars
- View dealers
- View reviews
- Render EJS pages
- Serve static CSS files
- Custom validation middleware
- Custom error-handling middleware

ROUTES: 

GET | '/api/cars' | Get all cars 

GET | '/api/cars/:id' | Get one car by ID 

GET | '/api/cars?make=ford' | Filter cars by make

POST | '/api/cars' | Create a new car |

PATCH | '/api/cars/:id' | Update a car 

DELETE | '/api/cars/:id' | Delete a car 


TECHNOLOGIES USED:

- Node.js
- Express.js
- EJS
- JavaScript
- HTML
- CSS
